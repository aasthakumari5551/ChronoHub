# ChronoHub

ChronoHub is an Employee Leave Management System with authentication, role-based access, and a modern UI using Node.js/Express and React.

## Features
- Employee leave requests
- Manager approval/rejection
- Admin user and leave policy management
- Authentication and role-based access
- Modern UI

## Project Structure
- chronohub-backend: Node.js/Express backend
- chronohub-frontend: React frontend

## Getting Started
1. Clone the repository
2. Install dependencies in both backend and frontend folders
3. Start backend and frontend servers
